# Rumi & The Way

*"As you start to walk on the way, the way appears."* — Rumi

## About

Brand brief presentation deck for **Rumi & The Way** — a philosophy-driven company built on intentional, discovery-led product building in the age of AI.

## View the Deck

Open `index.html` in any browser. Navigate with:

- **Click** anywhere to advance (left 20% of screen goes back)
- **Arrow keys** or **Space** to move forward/back
- **Home / End** to jump to first/last slide

## Slides

1. Title
2. The Origin — the Rumi quote that started it all
3. The Tension — speed vs. presence
4. The Philosophy — three core beliefs
5. Brand Identity — name, tagline, shorthand
6. Brand Voice — how we communicate
7. The Story You Tell — the elevator pitch
8. What's Next — first steps on the path
9. Closing

## Tech

Single-file HTML with Tailwind CSS (CDN), Playfair Display + Inter fonts, and vanilla JS for slide navigation. No build step required.
